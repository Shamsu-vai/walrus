# Walrus language
A tiny simple programming language made for simplicity. It borrows syntax from 'go', 'rust' and 'typescript'

[-] Lexer
[-] Parser
    [-] Variable declare
    [-] Variable assign
    [-] Array
        [-] Array indexing
    [-] User defined types
        [-] Struct
            [-] Property access
            [-] Property assignment
            [-] Private property deny access
        [-] Builtins
